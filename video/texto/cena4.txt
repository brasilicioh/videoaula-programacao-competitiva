# Exemplo de questão
Mostrarei agora uma questão desse esporte. Caso se interesse, leia com atenção e tente pensar em como resolvê-la.
Para resolver essa questão, basta você entender que o número de três das entradas será reduzido em um; após isso basta somar todos os valores.
Mostraremos agora três soluções possíveis em três linguagens de programação diferentes: Python, C++ e C.
O código em Python até pode ser o mais curto entre os três, mas não se deixe enganar por isso: as linguagens C e C++ são as mais usadas, pois são extremamente rápidas.
